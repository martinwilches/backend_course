# Subscription Tracker

## Generar la aplicacion de Express

```bash
npx express-generator --no-view --git ./
```

### Paquetes

#### nodemon

```bash
npm install --save-dev nodemon
```

#### eslint

```bash
npx eslint --init
```

#### dotenv

```bash
npm install dotenv
```

#### mongodb

```bash
npm install mongodb
```

#### mongoose

```bash
npm install mongoose
```
